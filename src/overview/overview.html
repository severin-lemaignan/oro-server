oro-server is a Java-based ontology server. It can load RDF/OWL ontologies and allow RPC access to the ontology. RPC framework is provided by external middleware. Currently only YARP is supported, but the server has been designed with modularity in mind and it should be fairly easy to add new ones.

The {@link laas.openrobots.ontology.IOntologyServer IOntologyServer} interface presents all the methods available to access and modify the ontology. The interface documentation provides detailled examples. Another source of example is the {@link laas.openrobots.ontology.tests.OpenRobotsOntologyTest OpenRobotsOntologyTest} class which contains about 15 unit tests, covering most of the API.
